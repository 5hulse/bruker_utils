Installation¶

bruker_utils is available via the Python Package Index. The latest stable version can be installed using:

$ python3 -m pip install bruker_utils

You need to be using Python 3.6 or above. bruker_utils also requires NumPy, which is installed automatically if you use pip install bruker_utils.
